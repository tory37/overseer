I have an idea that I would like to extend this app out to.

## The Concept
- I would like to add an optional layer on top of our agents.  
- The user should be able to, when creating a new instance, pick from a list of predefined pixel agents that add a new attitude to the ai.  
- the idea would be that, we add a small lightweight rule that defines a fun personality, and the ai should be instructed to act as this personality while interacting with the user.
- the personalities do NOT change their ability to code.  all agents should be world class coders, but this is a way to add a feel of managing different people.  it's purely in the way that they speak.
- ideally, we would put some kind of fun UI that makes it look like the guy is talking, with a speech bubble.  could be random characters that animate, or just ellipses like atext messge, if its possible to hook into this.
- It would be great if users could have a guided step by step process to make a new persona / underling / worker, whatever we want to call it.  and then we could store their custom ones locally.
- when opening a new agent, you assign a persona to work on it.  
- NOT selecting an persona, choosing no persona, would basically just work like it works now
